# Event Objekt
| Feld        | Datentyp           | Beschreibung  |
| ------------- |:-------------:| -----:|
| _id | Integer (Mongoose ObjectID) | Eindeutige Kennung |
| name | String | Name des Events |
| location | String | Ort des Events |
| date | UNIX Timestamp | Datum des Events |
| topic | String | Thema des Events |
